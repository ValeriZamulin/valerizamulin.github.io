О проекте

С каждым годом использование интернет ресурсов только растёт. Интернет стал нашей повседневной жизнью, поиск информации, шопинг, регистрация на приём к врачу и многие другие вещи делаются непосредственно через интернет. Для того, чтобы это всё возможно было осуществить, необходимо каждому предприятию либо организации наличие веб-сайта. Для значительного ускорения, упрощения, стандартизации дизайна и адаптивности веб приложений для всех устройств, при создании веб-сайта, разработчики используют различные фреймворки. Но какой же фреймворк выбрать? Таким вопросом задаются многие начинающие и некоторые продвинутые программисты. Различные виды веб-сайтов имеют индивидуальную структуру, из-за чего разработчик должен рассмотреть несколько фреймворков и определить наиболее подходящий, для решения поставленных задач. Время, которое тратит разработчик на выбор фреймворка зависит напрямую от опыта его работы в этой области.



Bootstrap

Это популярный фреймворк, который позволяет быстро и качественно создавать ста- тичные веб-сайты и веб-приложения. Это бесплатный набор инструментов, позволяющий использовать Html, CSS и JavaScript, разработанный компанией Twitter. Поддержка браузерами:
Chrome 45+
Firefox 38+
Safari 9+
Opera 30+
Edge 12+
IE10+

Materialize

Это новый фреймворк, который использует большое количество стилей, элементов, компонентов, анимации и, конечно, принципы Материального дизайна от Google. Есть две доступные версии – Standard, включающий в себя уменьшенные и не уменьшенные файлы CSS и JS, и версия SASS, предоставляющая Вам полную свободу выбора компонентов, которые Вы хотите использовать. Chrome 35+ Firefox 31+ Safari 9+ Opera Edge IE 11+

UIKit

представляет собой фраймворк с открытым исходным кодом, который позволяет веб сайту выглядеть хорошо. Легкий и модульный интерфейсный фреймворк подходящий для разработки быстрых и мощных веб-интерфейсов. На данной платформе есть возможность создавать свои собственные стили. UIkit дает Вам полный набор HTML, CSS, и JS компонентов, который является простым в использовании, легко настраивается и расширяемый. Chrome - Latest Firefox - Latest Safari 9.1+ Opera - Latest Edge - Latest IE 11+

О проекте

С каждым годом использование интернет ресурсов только растёт. Интернет стал нашей повседневной жизнью, поиск информации, шопинг, регистрация на приём к врачу и многие другие вещи делаются непосредственно через интернет. Для того, чтобы это всё возможно было осуществить, необходимо каждому предприятию либо организации наличие веб-сайта.


Для значительного ускорения, упрощения, стандартизации дизайна и адаптивности веб приложений для всех устройств, при создании веб-сайта, разработчики используют различные фреймворки. Но какой же фреймворк выбрать? Таким вопросом задаются многие начинающие и некоторые продвинутые программисты.


Различные виды веб-сайтов содержат различные компоненты и имеют индивидуальную структуру, из-за чего разработчик должен рассмотреть несколько фреймворков и определить наиболее подходящий, для решения поставленных задач.


Данный проект является практической частью заключительной работы ученика третьего курса Эстонского университета прикладных наук по предпринимательству Майнор, Валерия Замулина. Проект представляет собой веб приложение с интерактивным мастером, который поможет начинающим разработчикам при выборе фреймворка для создания сайта. В данном приложении подробно рассматриваются компоненты трёх популярных фреймворков: Bootstrap, Materialize и Uikit.



Bootstrap

Это популярный фреймворк, разработанный компанией Twitter, который позволяет быстро и качественно создавать статичные веб-сайты и веб-приложения.
Перейти на сайт фреймворка Bootstrap

Materialize

Это новый фреймворк, который использует большое количество стилей, элементов, компонентов, анимации и, конечно, принципы Материального дизайна от Google.
Перейти на сайт фреймворка Materialize

UIKit

Фраймворк с открытым исходным кодом, который позволяет веб сайту выглядеть хорошо. Легкий и модульный интерфейсный фреймворк подходящий для разработки быстрых и мощных веб-интерфейсов.
Перейти на сайт фреймворка Uikit
